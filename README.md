# NANOFIT - Virtual Outfit Try-on

This is a virtual clothing try-on application built with AI. To try out the app for free, use Google AI Studio (https://aistudio.google.com/app) which provides free API usage for image generation models within its platform.

## Get Your Free API Key

1. Visit Google AI Studio: https://aistudio.google.com/app
2. Create a free account and get your Gemini API key
3. Use this key to run the app locally

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (get one free at https://aistudio.google.com/app)
3. Run the app:
   `npm run dev`
